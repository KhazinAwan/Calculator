# Calculator

A web-based calculator to demonstrate the HTML, CSS and JavaScript fundamentails I learnt in the Odin Project's Foundations course.

##Features

- Operations: Addition, Subtraction , Multiplication and Division
- Supports chained calculations
- Will handle Decimal inputs
- Wil handle errors (e.g. division by zero)
- Will support keyboard inputs
- Will have delete and clear functionality
- Upon click/key press , the correct button changes its border color for visual feedback

##Keyboard Support

- 0-9 for digits
- "." for decimal input
- n for sign change
- c\C\Escape for clear
- Backspace for delete
- "+" for addition
- "-" for subtraction
- "*" for multiplication
- "/" for division
- Enter, "=" for evaluation

##Built with

- HTML
- CSS
- JavaScript
